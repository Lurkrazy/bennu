## Running

To run the code, execute the script `bennu/scripts/run_ansor.sh` to generate Ansor's template for a specific machine architecture. Default is x86-32.
The script `bennu/scripts/test_build_template.sh` executes 100 templates using Droplet Search as an optimizer of the Ansor's template.
